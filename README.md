# Portafolio Web - Mariajosé Escobar

Este proyecto corresponde a un portafolio web personal desarrollado con **HTML5** y **CSS3**.

## Descripción

La página presenta información personal, habilidades, proyectos y un formulario de contacto, utilizando una estructura clara, semántica y visualmente armónica.

## Secciones principales

- Barra de navegación
- Sobre mí
- Habilidades
- Proyectos
- Aside informativo
- Contacto
- Footer

## Tecnologías utilizadas

- HTML5
- CSS3
- Google Fonts

## Características

- Estructura semántica
- Diseño aesthetic
- Uso de Flexbox y Grid
- Paleta de colores personalizada mediante variables CSS
- Adaptación responsive para distintas pantallas

## Objetivo

El objetivo de este proyecto es mostrar mi avance en desarrollo web front-end, aplicando buenas prácticas de maquetación, diseño visual y organización del contenido.
